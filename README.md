# CNN Image Classification – CIFAR-10

This project implements a Convolutional Neural Network (CNN) using PyTorch to classify images from the CIFAR-10 dataset.

## Overview
- Built and trained a CNN model for image classification
- Applied data preprocessing and normalization
- Evaluated model performance on test data
- Visualized training results and predictions

## Technologies
Python, PyTorch, Torchvision, Matplotlib

## Dataset
CIFAR-10 dataset containing 10 image classes such as airplanes, cars, animals, and ships.

## Project Context
Originally implemented as part of the KAUST Advanced AI Training Program and later organized as a personal portfolio project.
